# Osmosis BHOP

A premium 3D browser-based Bunny Hopping (BHOP) game built using Three.js.

## Live Demo
Play the game live at: **[bhophub.github.io](https://bhophub.github.io)**

---

## Technical Architecture

*   **Core Engine**: Three.js (WebGL 3D Renderer)
*   **Physics**: Custom 3D kinematic movement, air acceleration, and collision detection
*   **Audio System**: Web Audio API with spatialized sound effects
*   **UI/UX**: Custom glassmorphism UI overlay built with modern HTML5/CSS3
*   **Branching & Deployment**:
    *   `main`: Public documentation and project portal (this branch).
    *   `gh-pages`: The deployed game source files served via GitHub Pages.
